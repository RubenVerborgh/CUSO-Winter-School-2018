Assignment: Semantic Web data modeling
and Linked Data interoperability

  1. Create your own RDFS/OWL ontology for photos.
    • title
    • author
    • people
    • date
    • description
    • …
  2. Link your ontology to the FOAF ontology.
  3. Describe a couple of photos with your ontology.
  4. SPARQL your photos using only FOAF. (source)
  5. Now run the same queries on others’ photos.
